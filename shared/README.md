# shared/

Codigo y assets reutilizables transversales a varias features.

| Carpeta | Contenido |
|---------|-----------|
| `ui/` | Componentes UI reutilizables (Sliders custom, paneles, themes) |
| `utils/` | Helpers GDScript: math, hash, paths, debug |
| `constants/` | Enums, paths, codigos de error, signal names |

Si algo se usa en una sola feature, NO va aca: va dentro de la feature.
